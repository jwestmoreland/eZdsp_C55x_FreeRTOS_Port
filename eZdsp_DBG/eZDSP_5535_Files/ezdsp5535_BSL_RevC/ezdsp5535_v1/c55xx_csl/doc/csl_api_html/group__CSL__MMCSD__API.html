Detailed Description

Introduction

Overview

MMCSD CSL provides set of APIs to configure and utilize the functionalities provided by the C5505/C5515 MMCSD hardware controller. C5505/C5515 MMCSD hardware controller provides interface for MMC,SD and SDIO cards. MMCSD CSL supports the MMC and SD and SDHC cards.

Terms and Abbreviations

  1. CSL: Chip Support Library
  2. API: Application Programmer Interface
