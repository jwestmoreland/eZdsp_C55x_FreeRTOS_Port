Overview

Universal serial bus is an interface using which USB devices can be interfaced to the host device. MUSB module is designed for the host-peripheral communication through different end points. It supports the operations like configuring the USB registers, Reading data from USB FIFO and writing data to USB FIFO etc. MUSB FCSL layer supports only peripheral mode functionality. Data can be transferred between the Host and the USB device either in DMA mode or in interrupt mode.
